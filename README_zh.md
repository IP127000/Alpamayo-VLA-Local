# Alpamayo-1-Local
VLA模型，英伟达Alpamayo 1的本地、离线运行版本。可在自己的机器上完全运行模型、处理数据并可视化结果。

## 使用方法
环境要求
```bash
python==3.12
cuda==12.1
```
```bash
git clone https://github.com/IP127000/Alpamayo-VLA-Local.git
```
```bash
cd Alpamayo-VLA-Local
```
```bash
pip install -r requirements.txt
```
```bash
python inference.py
```
